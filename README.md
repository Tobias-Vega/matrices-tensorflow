## Pasos necesarios para ejecutar el proyecto en cada uno de los directorios:

1. Clonar el repositorio:

   ```sh
   git clone https://github.com/Tobias-Vega/matrices-tensorflow
   ```

2. Instalar dependencias:

   ```sh
   npm install
   ```

3. Ejecutar el servidor:

   ```sh
   npm run dev
   ```
